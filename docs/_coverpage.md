# **Brainrot API**

- Provides random brainrot quotes, facts, dictionary words, names, and jokes.
- Free and opensource, can be self-hosted and self-deployed.
- Built using [Elysia](https://elysiajs.com/) and [Bun](https://bun.sh/).
